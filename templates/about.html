About MediCare

MediCare is a full-stack healthcare application developed to simplify the process of finding medicines for common symptoms. The platform features intelligent symptom-based search, dynamic recommendations, medicine management, and a responsive user interface, demonstrating practical implementation of web development and database integration concepts.

This project helps users find common medicines based on their health problems.
